# VCDcompare

## my setup

- Ubuntu 22.04
- python 3.10.12
- pip 22.0.2

## Getting Started
Install dependencies:
```bash
pip3 install -r requirements.txt    
```
Run the app:
```bash
python3 VCDcompare.py
```
It writes on a message.log file all the difference between changes of aimed signals from two vcd files
